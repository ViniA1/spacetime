## 🖥️ Projeto
Esse é um projeto responsivo de uma capsula do tempo que mostra minhas memorias em uma linha do tempo.

## 🛸 Tecnologia
Projeto desenvolvido durante o NLW da Rocketseat com as seguintes tecnologias:
- HTML
- CSS
- Git e Github
